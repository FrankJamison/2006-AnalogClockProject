# 2006 Analog Clock Project

A simple analog clock web application, inspired by classic designs, built with HTML, CSS, and JavaScript.

## Features
- Real-time analog clock display
- Smooth moving hour, minute, and second hands
- Responsive design for desktop and mobile
- Customizable styles via CSS

## Demo
To view the clock, open `index.html` in your browser or use the provided task to launch it locally.

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)

### Running Locally
1. Clone or download this repository.
2. Open `index.html` in your browser.
   - Or, if using the provided VS Code task, run the **Open in Browser** task to launch the project at [http://2006analogclockproject.localhost/](http://2006analogclockproject.localhost/).

## Project Structure
```
index.html         # Main HTML file
css/style.css      # Styles for the clock
js/script.js       # JavaScript for clock functionality
```

## Customization
- Edit `css/style.css` to change the appearance of the clock.
- Modify `js/script.js` to adjust clock behavior or add features.
